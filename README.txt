>>>> Cách chạy source code:
	1. Từ file source code, vào thư mục api
	2. Gõ từ "cmd" trên thanh đường dẫn
	3. Chạy dòng lệnh "npm install"
	4. Chạy dòng lệnh "npm start"
	5. Từ file source code, vào thư mục client
	6. Gõ từ "cmd" trên thanh đường dẫn
	7. Chạy dòng lệnh "npm install"
	8. Chạy dòng lệnh "npm start"

	Yêu cầu: Đã cài đặt NodeJS trước đó.
